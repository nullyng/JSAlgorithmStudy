# 문제 개요

**문제:** [문자열 나누기](https://school.programmers.co.kr/learn/courses/30/lessons/140108)

**출처:** 프로그래머스

**난이도:** Lv.1

# 문제 풀이

첫 문자를 변수에 저장하고 이후 문자부터 차례대로 아래를 수행한다.

1.  현재 문자의 개수가 첫 문자의 개수와 같다면
    - 정답을 증가시킨다.
    - 첫 문자를 현재 문자로 바꾼다.
    - 첫 문자 개수를 1로 설정한다.
    - 첫 문자를 제외한 문자의 개수를 0으로 설정한다.
    - 다음 문자로 넘어간다.
2.  그렇지 않다면 현재 문자와 첫 문자의 동일 여부를 판단해서 각 카운트 변수를 업데이트한다.
    - 현재 문자와 첫 문자가 같다면 첫 문자의 개수를 증가시킨다.
    - 현재 문자와 첫 문자가 다르다면 첫 문자를 제외한 문자의 개수를 증가시킨다.

마지막에 남는 문자열이 하나 있으므로 `answer+1`을 반환한다.
